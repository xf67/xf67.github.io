一、力学

A、运动学

  1. 放炮类：
    1. $\displaystyle \begin{cases} f( x,y,\theta ) =0\ \partial f/\partial \theta =0 \end{cases}$消θ，解$f(x,y)=0$便得到包络线
    2. 取斜面建坐标系还是以水平垂直
    3. 跨两个柱子，以从第一个柱子斜抛分析为易
    4. 两个都是抛体运动要相撞，取以$g$下降的系，则可直线分析
    5. 有阻力，linkB5
  2. 求空间中点的速度（eg.交点）：把函数写出来求导，只要把被交的两个函数两边求导然后解$Vx,Vy$就是了
  3. 圆套圆类：相对运动复杂，算相对算牵连。
    1. 注意在用$ω∗r$时，是哪个$r$， 会不会是$R−r$。
    2. A绕O'转，O'绕O转，A→O=(A→O′→O算相对)+(A→O算牵连)
    3. 科氏力，得判

    1. 运动规律相同无需分段，尽量合成整体考虑
    2. 等比数列

    4. 运动是否会有“相变”，link水平分量和竖直分量谁先衰减完的讨论

  3. $dv/dt=dv/dxdx/dt=vdv/dx$​​​常用代换 想法：实际只有两个变量的微分方程，换一换微元，使两边表面上也只有两个变量 经常用于：暴列微分方程，例如力和$x,v$​有关时
  4. 摩擦力的方向得判断，尤其在相对速度有两个方向的时候，$\mu N$​​怎么分配
  5. 取自然坐标系，但注意算出来F也要一路上的m之和
  6. 求曲率半径

  7. 限制条件大赏

    3. 轻杆的力可以不沿杆
    4. 铰链没有力矩，铰链上的杆在垂直杆方向合力矩为0，但可以提供垂直杆的合力
    5. 杆靠在圆周的一角，接触点的支持力方向垂直杆；杆的一角靠在圆周上，支持力指向圆心
  8. 边界条件大赏
    1. $N=0$​​​脱离（常见于圆弧面，为保持圆周运动需提供向心力，离心力=向心部分合力； 亦见于平面斜靠杆，取质心系，水平向惯性力（平动加转动）=水平其他合力）

C、基本守恒量

  1. 能量（机械能），动量，角动量，问问自己守恒的条件，以及取了什么系统
    1. 拐直角弯总怀疑有碰撞和能量损失，然桌角视为微小圆弧，可细算（link珍题），$r\rightarrow 0$可认为是能量守恒
    2. 一物将一坨绳带起来，算完全非弹性碰撞，有能量损失
    3. 轻绳摆过去，撞到一个钉子，摆长变化，能量不损失
  2. 变质量无内能有受力类（水滴下落吸附，球把绳子往上带）
    1. 列动量方程$d(mv)=F(v,x,t)dt$​
    2. 列$dm=f(t)dt$​​
    3. 将动量里的$dt$​​用$dm$​​替代，有$v$​就配个$m$​往左放，左边$mv$​整体积分，右边对$m$​积分，得到$v=v(m)$
    4. 再积一下$dm=f(t)dt$​,得到$m=m(t)$​换进$v$​去，得到$v=v(t)$​​
  3. 变质量有内能有受力类（火箭，光子火箭，水火箭）
    1. 计算“燃气”速度，注意是相对火箭还是相对地
    2. 列动量定理，或美称之曰“密舍尔斯基方程”，$mv-mgdt=(m+dm)(v+dv)-udm \Rightarrow mdv=-mgdt+dm(u-v)$​ 其中u是绝对速度，$u,v$​​​​皆以向上为正，dm以增大为正（故dm为负，小心了！），故若相对火箭速度恒定，单位时间喷质量亦恒定，则好积分矣
    3. 一般这时候就没法简单凑mv的全微分了，但是可以暴积加速度得速度，再积就是位移
  4. 有受外力之物难以分析，以之为系，取受系统内力之物对其动，分析较易
  5. 和一个环在里面碰碰碰，取环为系，初末态考究 （甚至有洛伦兹力也可以这么做，低速的系不影响B，至少能便利算碰撞的时间间隔，以及碰撞发生的方向）
  6. 小气的几样东西动来动去类
    1. 质心没有位移？水平动量守恒？
    2. 二体碰撞？取质心系，最大可能能量损失为资用能。
    3. 换系，考虑各种速度关联？
  7. 天体运动各类常用结论&模型
    1. 比耐方程，用$\omega$​​​​​里的dt来代替dt，使方程不含t，用u=1/r代替r，用L守恒量，写牛二，记得算$\dot{r}$和$\ddot{r}$时要一个个算循序渐进，并且时刻记得用L守恒量去把东西代掉，否则就不好推。后带入F的形式，平方反比，可迅速得出椭圆。
    2. 开普勒定律，很基本，但是好用
      1. 二体质量差不大时，绕质心转，如何考虑等效需细想，不是约化质量，R～Rc的变化是平方的，把万有引力完整写出来才好作类比
      2. 直线看成椭圆的极限，另外注意来回的问题
      3. 用面积速度的时候注意三角形面积要除以2
    3. 用好有心力L守恒，注意L=mvr的v只是切向速度，以及思索什么时候的v是只有切向的（转折点，或者说r极值点）
    4. 总能量$E=-\frac{GMm}{2a}$​​​（为负号表示椭圆运动），=0​则抛物线，正号为双曲线
    5. 引力弹弓，其好处就是在于在木星参考系下V⊥和V∥（V∥相对日-V木），保持能量守恒下变得只有V∥（与木星轨道相切的方向），换回太阳参考系又能再叠上V木，使绕日的轨道速度大增。实际上是白嫖了木星的一部分轨道能量，只是对于木星微不足道。
    6. 有效势能$E=U(r)+{L^2}/{2mr^2}$​​​​​​​，即考虑进了惯性离心势能（它是正的！） 此可用于算轨道进动，以$F=dE/dr$​算回复力（令$r=r_0+\delta$​，用$\delta$算），得到径向振动的$\omega_r$，进动角速度$\omega_p =\omega_r -\omega_o$​​​，乃长轴转动的角速度，与原轨道转动方向相反为正
  8. 各种坑大赏
    1. 选取势能零点（引力势能，在气团时；电学里更坑，link珍题），势能类坚定地以无穷远为0，其他类定好标准后不要乱
    2. 运动有“相变”，eg.会脱离作斜抛
    3. 万有引力，气团或者星体打洞，M会变
  9. 基本做题步骤
    1. 想象情景，判断守恒量
    2. 列初态和末态
    3. 判断几何关系，限制条件，边界条件
    4. 解微分方程

F、振动波动

  1. 求小振动的频率
    1. 找到单自由度的合适坐标（建议：先写多自由度的，再写自由度之间的关联限制方程，不要一次性取一个广义坐标，易错）
    2. 写$E_k(x^2)+E_p(\dot{x})=Const$​​，求导得$\ddot{x}=-\omega ^2 x$，即可
  2. 简振模
    1. 我赌不会考
    2. 写出多个自由度下的多个牛顿第二定律
    3. 设这些自由度都是$x_i=A_i cos(\omega t)$，代入上面牛二，得到一组$f(\omega,A_i)=0$​
    4. 整理为一组$\Sigma k_i A_i=0$的形式
    5. 以A为变量，这个线性方程的系数行列式要求为0，解出$\omega$​的多个可能解，每个$\omega$​都是一个可能的简谐振动模式
    6. 如果几个自由度之间还不独立，则需找到一套相互独立的简正坐标……
  3. 阻尼振动 $\ddot{x}+2\beta \dot{x}+\omega_0^2x=0 \Rightarrow $​​特征方程$x^2+2\beta x +\omega_0^2=0$​，$\Delta=4(\beta^2-\omega_0^2)$​​​​，解为$x_1,x_2$​
    1. $\beta=\omega_0 \Rightarrow x=e^{-\beta t}(A+Bt)$​ 临界阻尼
    2. $\beta>\omega_0\Rightarrow x=Ae^{-x_1t}+Be^{-x_2t}$​​过阻尼
    3. $\beta<\omega_0\Rightarrow x=Ae^{-\beta t}cos(\omega t+\varphi)$​欠阻尼
  4. 受迫振动 $\ddot{x}+2\beta \dot{x}+\omega_0^2x=h*cos(\omega t+\varphi_1) $​​​存在相位是因为主动力也将有收到阻尼
    1. 解为$x=Bcos(\omega t+\varphi_1+\varphi_2)$
  5. 重弹簧
    1. 用两套坐标，$l/l_0=u/u_0$​，$l$​是现总长，$l_0$是原长，$u$是现长对应位置，$u_0$​​是原长对应位置
    2. $k\frac{L}{dx}du=F,dF=dm*\ddot{u}=m\frac{dx}{L}\ddot{u} \Rightarrow \frac{\partial^2u}{\partial x^2}-\frac{m}{kL^2}\frac{\partial^2u}{\partial t^2}$​，得到波动方程，其中波速$v=m/(kL^2)$​
    3. 传统能量法，Ep还是$1/2*ku^2$​​，Ek增加弹簧的动能项，对于求重弹簧振子的频率还是很方便
  6. 简谐波
    1. 波动方程$\frac{\partial^2u}{\partial x^2}-\frac{m}{kL^2}\frac{\partial^2u}{\partial t^2}$其解为$u(x,t)=A(x)cos(\omega t+\varphi)=Acos(\omega x/v+\varphi_A)cos(\omega t+\varphi)$​
    2. 各种波速：绳波$v=\sqrt{T/\rho}$​​​ 声波 ……
    3. 能流：$S=1/2 *\rho v \omega ^2A^2$
    4. 多普勒公式，上接下发，速度分解到相对位移方向，靠近则频率升

      6. 于介质界面$\sigma_自=D_n,\sigma_极=-P_n,\sigma_总=\sigma_自+\sigma_极=\varepsilon_0E_0$​
      7. 对于介质界面与等势面重合的情况，算出没有电介质的E，有电介质处除以$\varepsilon_i$即可
      8. 对于介质界面和等势面垂直的情况，算出没有电介质的E，有介质的部分设为$\alpha E$​，再用D的高斯定理通过Q定出$\alpha$

4.3Fresnel-Kirchoff衍射积分:$ \overset{{ \sim }}{U}( P) =\frac{-i}{2\lambda }\iint ( cos\theta {0} +cos\theta )\overset{\sim }{U}( Q)\frac{e^{ikr}}{r} d{\textstyle \sum }$ 字符声明:S为光源,在波面$ \sum $上一点Q,外还有一点P,$ d\sum $的法向为$ \mathbf{n}$. 4.4Babinet原理:互补屏造成的衍射场中复振幅之和等于自由波场的复振幅.即可以方便地由互补衍射屏的衍射图样得出原衍射屏的衍射图样.

4.5.1半波带法⟹分割波前为一系列环带到P逐次差半个波长⟹自由传播时整个波前在$ P{0}$的振幅是第一个半波带的效果的一半.
4.5.2矢量图解法⟹分成更细的波带,相邻环带到P差$ \pi /m$的相位,将A首尾相接方向逐个旋转$ \pi /m$,然后看合振动为何
4.5.3菲涅耳波带片:半波带半径$ \rho {k} =\sqrt{\frac{Rb\lambda }{R+b} k} \ ,k\in Z^{+}$,若是平行光则$ R\rightarrow +\infty $.(S波源,O波面顶点,P波外一点,SOP共线,波面上一点M,SM=R,MP=r,OP=b,MO$ \approx \rho $,傍轴近似)将奇数/偶数半波带涂黑,则制成了菲涅耳波带片,其使轴上距离波带片$ f=\rho ^{2}{1} /\lambda $处光强增加多倍,另外$ f/3,$ $ f/5,$ $ f/7\dotsc $的光强也将有所增大. 有近似透镜成像的公式$ 1/R+1/b=1/f$.
4.6夫琅禾费衍射-距离无穷远(菲涅耳衍射的极限)
4.6.1单缝衍射的强度:设像屏上正中的光强为$ I{0}$,在与光轴呈$ \theta $的光线所指点P的光强为$ I_{\theta }$,$ I_{\theta } =I_{0}( sin\alpha /\alpha )^{2} ,\alpha =\pi asin\theta /\lambda $,$ a$是缝宽.称$ ( sin\alpha /\alpha )^{2}$为强度因子
4.6.2矩孔衍射的强度:$ I_{P} =I_{0}( sin\alpha /\alpha )^{2}( sin\beta /\beta )^{2}$,即乘上两个方向的强度因子
4.6.3一些讨论:单缝衍射的零级衍射斑即几何光学的像点,亮斑角宽度$ \Delta \theta =\lambda /a$,但零级亮斑宽度为此再乘以2

4.8光栅/多缝夫琅禾费衍射
4.8.1光栅常数$ d=a+b$,$ a$为缝宽,$ b$为不透明部分宽
4.8.2 N缝衍射$ I{\theta } =A^{2}_{0}( sin\alpha /\alpha )^{2}( sinN\beta /sin\beta )^{2}$ ,$ \alpha =\pi asin\theta /\lambda ,\beta =\pi dsin\theta /\lambda $其中$ ( sin\alpha /\alpha )^{2}$是单缝衍射因子,$ ( sinN\beta /sin\beta )^{2}$是缝间干涉因子.各个极大的半角宽度$ \Delta \theta =\lambda /Ndcos\theta {k}$. 单缝衍射因子还将造成缺极,以及影响极大值的强度.
4.8.3正弦光栅和黑白光栅(好烦啊懒得看了)
4.8.4光栅公式$ sin\theta =k\lambda /d$,对应不同波长$ \theta $不同,故可用于分光
4.8.5角色散本领$ D{\theta } \equiv \delta \theta /\delta \lambda =k/( dcos\theta {k})$ , 线色散本领$ D{l} \equiv \delta l/\delta \lambda =fk/( dcos\theta _{k})$ ,设光栅后聚焦透镜焦距为$ f$,则有$ \delta l=f\delta \theta $
4.8.6色分辨本领$ R\equiv \lambda /\delta \lambda =kN$ , $ \delta \lambda $为能分辨的最小波长差
4.8.7光栅使用要求:$ \lambda _{max} < d$,最大波长小于光栅常数;$ \lambda _{min} >\lambda _{max} /2$
4.8.8闪耀光栅,不将能量全集中在中央主极大而是集中在1级或更后面的条纹上. 槽面法向与光栅宏观平面法向的夹角为$ \theta _{b}$,叫闪耀角. 在垂直槽面入射的时候$ 2dsin\theta _{b} =k\lambda _{kb}$,在垂直宏观面入射的时候$ dsin2\theta _{b} =k\lambda _{kb}$:将能量集中在$ \lambda _{kb}$光波的k级条纹上.
4.8.9三维光栅-X射线在晶体上衍射:Bragg条件-对于一个晶面族(一个晶体可有多个晶面族),只有满足$ 2dsin\theta =k\lambda $ ($ d$是晶格组成的面之间的间距),才能有主极大
4.8.10当用连续谱X射线照射单晶体,每个晶面族都从中选出符合自己布拉格条件的$ \lambda $来,主极大处出现亮斑,这样的现象称劳厄相.当用单色的射线照在多晶粉末上时,摆放方向不同的晶体亦选出了符合它们布拉格条件的方向,产生亮斑,这样的现象叫得拜相.
